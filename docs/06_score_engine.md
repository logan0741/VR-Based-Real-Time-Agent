# 06 — ScoreEngine

## 책임

DTW 거리 행렬과 RepDetector 구간 목록을 입력받아 실시간 점수와 운동 1회 점수를 계산한다.
웹캠 입력이 들어오는 동안 계속 계산 및 출력한다.

---

## 입출력 스펙

| 항목 | 내용 |
|------|------|
| 입력 | DTW 거리 행렬 shape `(M, K)` dtype `float32` |
| 입력 | RepDetector 구간 목록 `list[tuple[int, int]]` |
| 입력 | `weights`: 관절별 가중치 (config.py에서 읽어옴) |
| 입력 | `max_distance`: 선형 변환 기준 최대 거리 (config.py에서 읽어옴) |
| 입력 | `n_frames`: 실시간 점수 산출에 사용할 프레임 수 (config.py에서 읽어옴) |
| 출력 | 실시간 점수: `int` (0~100, n프레임 평균) |
| 출력 | 1회 점수 목록: `list[int]` (n번째 1회 점수 순서대로 누적) |

### 점수 산출 방식
- **실시간 점수**: 최근 n프레임의 가중 평균 거리를 선형 변환 (거리 0 → 100점, max_distance 이상 → 0점)
- **1회 점수**: RepDetector 구간 내 전체 프레임의 가중 평균 거리를 선형 변환
- 1회 점수는 메모리에 리스트로 누적 보관하며 파이프라인 결과 dict의 `rep_scores` 필드로 반환
