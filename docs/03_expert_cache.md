# 03 — ExpertPoseCache

## 책임

전문가 영상 전체 프레임에 PoseEstimator와 PoseNormalizer를 적용하여 정규화된 관절 시퀀스를 생성하고, 앱 시작 시 1회만 계산하여 메모리에 보관한다.

---

## 입출력 스펙

| 항목 | 내용 |
|------|------|
| 입력 | 전문가 영상 경로 (config.py에서 읽어옴) |
| 입력 | `target_fps`: 전문가 영상에서 추출할 초당 프레임 수 (config.py에서 읽어옴) |
| 출력 | 정규화된 관절 시퀀스 shape `(N, 17, 3)` dtype `float32` (N: target_fps 기준 샘플링된 총 프레임 수) |

---

## 완료 기준

전문가 영상 전체를 처리했을 때 프레임 수(N)가 1 이상이고, 각 프레임의 관절 좌표가 정규화된 상태인지 확인.
